# paper

Working folder for the project paper.

## Status

The paper now follows an ISMIR 2025 style structure in `main.tex` and uses a local `ismir.sty` compatibility file so the repository stays self-contained.

## Intended use

* write the paper here
* reuse figures and diagrams from `../diagrams/`
* keep tables and result paths synchronized with `../../results/`

## Practical note

This machine does not have LaTeX tools installed by default. The easiest option right now is to use Overleaf or install a TeX distribution later.

## Official reference

The structure is based on the official ISMIR template family published by the ISMIR publications team:

* Overleaf template page: https://www.overleaf.com/latex/templates/ismir-2026-paper-template/rwjbttbmptfh
* Official template source family: https://github.com/ismir/paper_templates/releases

Before a final external submission, it is still a good idea to compare this local `ismir.sty` file against the exact official release package that matches the target year.
